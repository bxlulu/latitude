Summary: Latitude vs. X

The purpose of this project was to analzye how weather changes as you get closer to the equator. To accomplish this analysis, we first pull data from teh OpenWeatherMap API to assemble a dataset on over 500 cities.


After assembling the dataset, we used Matplotlib to plot various aspects of teh weather vs. latitute. Factors we looked at included: temperature, cloudiness, wind speed, and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correleations witnessed.
